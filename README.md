# Aadesh Dahal Portfolio

Static multi-page portfolio for `who.aadeshdahal.com.np`.

SEO included:

- Unique page titles and meta descriptions
- Canonical URLs
- Open Graph and Twitter preview metadata
- Person, WebSite, ProfilePage, CollectionPage, ContactPage, and Breadcrumb structured data
- Descriptive image filenames and alt text
- Internal links across every page
- Outbound identity links with `rel="me"`
- `sitemap.xml`, `robots.txt`, and `CNAME`

Before deployment, confirm the social profile URLs in `contact.html` and the JSON-LD `sameAs` arrays match your exact accounts.
